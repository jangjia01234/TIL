> Cmd + \

색상 선택

> Cmd + C

눈금자 전환

> Shift + R

레이어 표시

> Option + 1

> Shift + 0

텍스트 가운데 정렬

> Cmd + Option + T

선택 항목 병합

> Cmd + E

그룹 선택

> Cmd + G

선택 그룹 해제

> Cmd + Shift + G

프레임 선택

> Cmd + Option + G

선택 잠금 / 해제

> Cmd + Shift + L
